- Groups must be created in the Cloud Access Manager (CAM), they cannot be created via the Elektra dashboard.
- Users can assign themselves to groups directly through CAM, following approval workflows and audit trails for full compliance.
- In CAM, keystone groups are represented as CAM Access Levels assigned to CAM Profiles, which users request access to.
- Request a CAM Access Level (and optionally a CAM Profile) if group membership changes must follow security or compliance processes.
